# 云函数环境变量配置说明

## 配置步骤

1. 登录微信云开发控制台
2. 进入「云函数」→ 选择 `getLofData`
3. 点击「配置」标签
4. 在「环境变量」中添加：

| 变量名 | 变量值 | 说明 |
|--------|--------|------|
| `LOF_API_URL` | `http://154.8.205.159:8081` | LOF Monitor API 地址 |
| `LOF_API_TOKEN` | `你的API Token` | 从 keys/小程序.txt 获取 |

## 注意事项

- ⚠️ **不要**将实际的 Token 提交到 Git 仓库
- Token 信息保存在本地 `keys/小程序.txt` 文件中
- 该文件已在 `.gitignore` 中排除
